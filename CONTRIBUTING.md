# Contributing

This is a focused argument: AI broke Principle 7 of the Agile Manifesto, and it needs fixing. Contributions should engage with that argument.

## Propose changes

Open an issue to discuss the change before submitting a pull request. This is a short document where every word matters, so discussion first helps avoid wasted effort.

## Translate

Translations are welcome. Add them to a `translations/` folder using the format `amendment.[language-code].md` (e.g., `amendment.es.md`).

## Guidelines

- Keep proposals concise. The amendment's strength is its brevity.
- Ground arguments in experience, not theory.

## What this repo is not

This is not a general forum for AI-and-software-engineering discussion. Contributions should connect to the specific argument about Principle 7 and its implications.
