# Contributing

This addendum is a living document. It emerged from a specific argument about how AI is
changing what software engineering processes need to produce, and contributions should
engage with that argument.

## Sign the addendum

If you agree with the values and principles in this addendum, add your name to [SIGNATORIES.md](SIGNATORIES.md) via pull request.

Include your name, title/role, and optionally a link to your profile.

## Propose changes

Open an issue to discuss the change before submitting a pull request. This is a short document where every word matters, so discussion first helps avoid wasted effort.

## Translate

Translations are welcome. Add them to a `translations/` folder using the format `manifesto-addendum.[language-code].md` (e.g., `manifesto-addendum.es.md`).

## Guidelines

- Keep proposals concise. The addendum's strength is its brevity.
- Ground arguments in experience, not theory.
- Respect the structure of the original Agile Manifesto.

## What this repo is not

This is not a general forum for AI-and-software-engineering discussion. Contributions
should connect to the specific framework and values proposed here.
